# PDF_Quotient_Code
This code estimates the quotient of probability density functions (PDF) between two time series, following the approach originally proposed by Xu et al. (2007).
It is associated with the study of Robache & Schmitt (2025), published in <i>Biogeosciences</i> (Copernicus journal).

If you use this code, please cite the following references:
- Xu, H., Ouellette, N. T., and Bodenschatz, E.: Curvature of Lagrangian Trajectories in Turbulence, <i>Physical Review Letters</i>, 98, 050 201, https://doi.org/10.1103/PhysRevLett.98.050201, 2007
- Robache, K. and Schmitt, F. G.: Multiscale statistical analysis of thermal and non-thermal components of seawater <i>p</i>CO<sub>2</sub> in the Western English Channel: scaling, time-reversibility, and dependence, <i>Biogeosciences</i>, 22(24), 8093-8111, [https://doi.org/10.5194/egusphere-2025-972](https://doi.org/10.5194/bg-22-8093-2025), 2025
